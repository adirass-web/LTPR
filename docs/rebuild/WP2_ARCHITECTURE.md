# WP2 — Bilingual architecture

**Status:** In progress

## Scope

- Shared EN/HE page shell, navigation, footer, and neutral page frame.
- Eight contracted locale routes plus a root default-language redirect.
- Per-route `lang`, `dir`, canonical URL, `hreflang`, locale-aware metadata, and paired language switching.
- Logical Tailwind layout utilities only; no physical directional positioning.
- `<bdi>` isolation for the Latin author name in Hebrew structure.

## Explicitly deferred

- WP3 visual identity, typeface selection, palette implementation, photography direction, and interaction refinement.
- WP4 copy migration and real page composition.
